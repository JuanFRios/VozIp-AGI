Para ejecutar esta aplicación debe realizar los siguientes pasos:

1. Debe instalar :
Apache para Windows.
PHP4.0
MySQL
Myadmin.

Puede encontrar información acerca del proceso de instalación  en las siguientes direcciones:
www.php.net
www.apache.org
www.neuroweb.org
www.mysql.com
www.phpwizard.net

2. Luego de la instalación, utilizando myadmin  debes crear una base de Datos llamada Biblioteca y crear en ella siguiente tabla.

CREATE TABLE libros (libro INT NOT NULL AUTO_INCREMENT, nombre CHAR(100), autor CHAR(50), tema CHAR(15), comentario CHAR(100), KEY (libro) );

La función mysql_connect("pc-140.red -udea", "root");  conecta  a la base de datos del servidor pc-140.red-udea., se debe especificar el servidor y password en el cuál se encuentra la base de datos.
